  Main logo image currently used in the top-level `README.md`.

## Usage

- Use `pitlane-mcp-logo.png` for the repository landing page and README.

- Keep branding assets in this directory instead of scattering them across the repo.
- `pitlane-mcp-logo.png` is currently the only branding asset in the repo.
- No separate light-mode, dark-mode, icon-only, or social-preview variants are currently available.
- If the README changes branding presentation, update this page so the expected asset remains clear.
